# PHP Index Dashboard 포트 관리 및 사용법

## 🚀 포트 3012 서버 관리 (Windows)

### ✅ 서버 시작
더블클릭으로 실행:
```
start-server.bat
```

**자동 처리:**
- 포트 3012가 이미 사용 중이면 자동으로 기존 프로세스 종료
- Node.js 설치 여부 확인
- 서버 자동 시작

### ⛔ 서버 중지
더블클릭으로 실행:
```
stop-server.bat
```

**자동 처리:**
- 포트 3012의 모든 프로세스 종료
- 안전한 종료 확인

---

## 🔧 색인 관리 패널 기능

### 1️⃣ 색인 정보 확인
대시보드의 "🔧 색인 관리" 탭에서 다음 정보를 확인할 수 있습니다:

**위치 정보:**
- 🎯 **색인 파일 위치**: 생성된 인덱스 JSON 파일 경로
- 📂 **소스 폴더 위치**: 색인된 PHP 소스 코드 경로

**상세 정보:**
- 💾 파일 크기
- 📈 심볼 수 (함수, 클래스 등)
- ⏰ 생성 시간
- 📊 커버리지

---

### 2️⃣ 소스 폴더 변경 및 재색인

**기능:**
새로운 PHP 소스 폴더를 지정하여 해당 폴더의 코드를 색인화합니다.

**사용 방법:**

1. **📂 소스 폴더 변경** 섹션에서 경로 입력
   ```
   work/mobile
   /path/to/php/files
   C:/projects/my-app
   ```

2. **"경로 변경 및 재색인"** 버튼 클릭

3. 확인 대화상자에서 "확인" 선택

4. 진행률 바 표시 → 완료 후 자동 갱신

**예시:**
```
입력: work/ecommerce
결과: work/ecommerce 폴더의 모든 PHP 파일 색인화
```

---

### 3️⃣ 현재 경로 재색인

**기능:**
기존 설정된 경로의 색인을 다시 생성합니다.

**사용 방법:**
1. **🔨 관리 도구** 섹션의 "현재 경로 재색인" 버튼 클릭
2. 확인 대화상자에서 "확인" 선택

---

### 4️⃣ 캐시 삭제

**기능:**
생성된 캐시 파일을 모두 삭제합니다.

**사용 방법:**
1. **"캐시 삭제"** 버튼 클릭
2. 확인 대화상자에서 "확인" 선택

---

### 5️⃣ 색인 내보내기

**지원 포맷:**
- 📄 JSON: 원본 인덱스 데이터 (프로그래머용)
- 📋 CSV: 스프레드시트 호환 형식

**사용 방법:**
1. **"내보내기"** 버튼 클릭
2. JSON 또는 CSV 선택
3. 파일 다운로드

---

## 📊 통계 정보

색인 생성 후 다음 통계를 자동으로 표시합니다:

| 항목 | 설명 |
|------|------|
| **함수** | 발견된 PHP 함수 개수 |
| **클래스** | 발견된 클래스 개수 |
| **파일** | 색인된 PHP 파일 개수 |
| **라인** | 총 코드 라인 수 |
| **인코딩** | 파일 인코딩 (UTF-8) |
| **버전** | 색인 버전 |

## ⚠️ 주의사항

### 포트 3012가 이미 사용 중일 때
```
# 1. stop-server.bat 실행
# 2. 또는 수동으로 종료:
netstat -ano | findstr :3012
taskkill /PID [PID] /F
```

### 소스 폴더 경로 형식
- ✅ 상대 경로: `work/mobile`, `../php-files`
- ✅ 절대 경로: `C:/projects/app`, `/usr/local/www`
- ❌ Windows 백슬래시는 피함: `C:\projects\app` → `C:/projects/app`

### 재색인 시간
- 파일 개수에 따라 몇 초~수 분 소요
- 진행률 바로 진행 상황 확인 가능

## 📝 문제 해결

### 서버가 시작되지 않음
- Node.js 설치 확인: `node --version`
- 포트 3012 확인: `netstat -ano | findstr :3012`

### 색인이 생성되지 않음
- 경로 존재 확인
- PHP 파일이 실제로 있는지 확인
- 권한 확인 (읽기 권한 필요)

### 색인이 오래되었을 때
- 색인 관리 패널에서 재색인 실행
- 캐시 삭제 후 재색인
