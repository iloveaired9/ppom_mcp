# 뽐뿌 자유게시판 실시간 분석

> 최신 freeboard 데이터를 자동으로 수집하고, 분석하여 요약 리포트를 생성합니다.

## 🚀 사용 방법

### 기본 사용 (1페이지)
```bash
/fetch-and-summarize-freeboard
```

### 특정 페이지 분석
```bash
/fetch-and-summarize-freeboard 2
/fetch-and-summarize-freeboard 3
```

### 여러 페이지 연속 분석
```bash
/fetch-and-summarize-freeboard 1 2 3
```

---

## 📊 동작 과정

```
1️⃣  MCP 호출
   └─ GET http://localhost:3008/freeboard?page=N
      └─ 최신 게시물 30개 데이터 수집

2️⃣  CSV 자동 저장
   └─ freeboard-{YYYY-MM-DD}-page{N}.csv
      └─ 분석 추적용 기록

3️⃣  Claude 자동 분석
   └─ /analyze-freeboard 스킬 호출
      └─ 카테고리, 키워드, 트렌드 분석

4️⃣  요약 리포트 생성
   └─ 마크다운 형식으로 시각화
      └─ 차트, 통계, 인사이트 포함
```

---

## 📋 분석 내용

### 🎯 핵심 요약
- 총 게시물 수
- 분석 시간
- 최상위 카테고리
- 최활발 시간대

### 📊 카테고리별 분포
- 정치/검찰 개혁
- 국제 이슈
- 스포츠
- 연예/종교
- 경제/쇼핑
- 생활 이야기

### 🔥 핵심 키워드 (TOP 5~10)
- 제목에서 자동 추출
- 빈도 기반 순위
- 관련 주제 분석

### 🏆 상위 게시물 (TOP 5)
- 조회수 기준
- 추천수 기준
- 참여도 기준

### 💬 참여도 분석
- 평균 추천수
- 최대 추천수
- 조회수 분포
- 활발도 판정

### ⏰ 시간대 분석
- 시간대별 게시물 집중도
- 활발한 시간대 탐지
- 패턴 분석

## 🔄 비교 분석

이전 분석과 비교하여 자동 감지:

### 📈 증감 추적
```
정치/검찰: 40% → 50% ⬆️ (+10%)
경제/투자: 0% → 17% ⬆️ (신규 이슈)
스포츠: 10% → 13% ⬆️ (+3%)
```

### 🔍 새 키워드 탐지
- **신규**: 오세훈, 현대차
- **약화**: 호르무즈 해협

## 🛠️ 내부 구동 원리

### Step 1: 데이터 수집
```bash
curl -s http://localhost:3008/freeboard?page={page}
→ JSON 응답
→ posts 배열 추출
```

### Step 2: CSV 변환
```javascript
{
  no, title, author, date, views, recommend
}
→ CSV 행으로 변환
→ freeboard-{date}-page{N}.csv 저장
```

### Step 3: 분석 스킬 호출
```bash
/analyze-freeboard freeboard-{date}-page{N}.csv
→ 카테고리 분류
→ 키워드 추출
→ 통계 계산
→ 마크다운 리포트 생성
```

## 🚨 주의사항

1. **MCP 서버 필수**: ppomppu-crawler MCP 서버가 3008 포트에서 실행 중이어야 함
   ```bash
   # 확인
   curl http://localhost:3008/health
   ```

2. **CSV 파일 관리**: 많은 페이지 분석 시 CSV 파일 증가
   ```bash
   # 정리
   rm freeboard-*.csv
   ```

3. **분석 시간**: 첫 실행 시 약 3~5초 소요
