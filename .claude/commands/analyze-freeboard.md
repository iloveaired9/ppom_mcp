# 뽐뿌 자유게시판 현황 분석

> freeboard.csv 파일의 게시물을 분석하고 카테고리별/키워드별 현황을 마크다운 리포트로 생성합니다.

## 📋 분석 항목

1. **카테고리 분류** - 정치/검찰, 국제, 스포츠, 연예/종교, 경제/쇼핑, 생활 등
2. **핵심 키워드** - 제목에서 자주 나타나는 주요 단어 (TOP 5)
3. **상위 게시물** - 조회수/추천수가 높은 게시물 (TOP 5)
4. **참여도 분석** - 평균 추천수, 최대값, 활발도
5. **시간대 분석** - 언제 게시물이 집중되어 있는가?

## 🚀 사용 방법

```
/analyze-freeboard freeboard.csv
```

또는 기본값 사용:
```
/analyze-freeboard
```

## 분석 로직

### 1️⃣ 카테고리 분류 규칙

```
정치/검찰: 검찰, 대통령, 윤석열, 이재명, 여당, 야당, 민주당, 검찰개혁
국제: 호르무즈, 이란, 미국, 이스라엘, 중국, WBC, 해협
스포츠: 야구, 팀, 경기, 선수, 경기, 승리, 패배, WBC
연예/종교: 김어준, 신천지, 영화, 드라마, 반지의제왕
경제/쇼핑: 포인트, 네이버, 페이, 구매, 쿠폰
생활: 일상, 고민, 의견, 이야기
```

### 2️⃣ 키워드 추출

제목의 명사/고유명사 추출 후 빈도 순 정렬

### 3️⃣ 참여도 계산

- 추천수 열의 `X - Y` 형식 파싱
- 조회수 기반 순위 계산
- 평균/최대값 산출

### 4️⃣ 시간대 분석

`작성시간` 열의 시간(HH) 기반으로 분류

## 📈 해석

### 정치 이슈가 압도적
- 검찰개혁이 가장 많은 언급
- 여당/야당, 인물 관련 논쟁 활발

### 국제 분쟁 이슈
- 호르무즈 해협 기뢰 설치 이슈 (외교 위기)
- 이란-미국-이스라엘 긴장 반영

### 스포츠 및 연예
- WBC 야구 및 성적 논의
- 김어준 관련 정치/사회 논쟁

### 높은 참여도
- 사회 문제 관련 게시물에 높은 추천수
- 정치 이슈 관련 댓글 활발

## 🔄 확장 가능성

이 분석은 다음과 같이 확장 가능합니다:

1. **다른 게시판**: `/analyze-board {board-name}`
   - baseball, stock, entertainment, etc.

2. **시간대별 추세**: `/analyze-trend {board} {days}`
   - 7일, 30일 추이 분석

3. **비교 분석**: `/compare-boards {board1} {board2}`
   - 두 게시판의 주제 비교

4. **감정 분석**: `/sentiment-analysis {board}`
   - 긍정/부정/중립 비율
