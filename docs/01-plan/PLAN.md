# Todo App 기능 계획

## 프로젝트 개요
간단하면서도 실용적인 Todo 애플리케이션을 만드는 프로젝트입니다.
MCP 학습을 통해 완전 자동화된 개발 워크플로우를 구축합니다.

---

## 📋 기능 목록

### Phase 1: MVP (필수 기능)
| # | 기능 | 설명 | 우선순위 |
|---|------|------|---------|
| 1 | 할 일 추가 | 새로운 할 일 항목 생성 | 높음 |
| 2 | 할 일 목록 | 모든 할 일 표시 | 높음 |
| 3 | 할 일 완료 표시 | 완료 상태 토글 | 높음 |
| 4 | 할 일 삭제 | 할 일 항목 제거 | 높음 |

### Phase 2: 고급 기능
| # | 기능 | 설명 | 우선순위 |
|---|------|------|---------|
| 5 | 할 일 카테고리 | 카테고리별 분류 | 중간 |
| 6 | 우선순위 설정 | 높음/중간/낮음 | 중간 |
| 7 | 마감일 설정 | 기한 추가 | 중간 |
| 8 | 검색 기능 | 할 일 검색 | 낮음 |

## 🎯 기술 스택

### Frontend
- **Framework**: Next.js 14 (App Router)
- **스타일**: Tailwind CSS
- **상태 관리**: React Context / Zustand

### Backend
- **서버**: Node.js + Express / Fastify
- **데이터베이스**: PostgreSQL
- **ORM**: Prisma

### DevOps
- **버전 관리**: Git
- **CI/CD**: GitHub Actions
- **배포**: Vercel (Frontend) / Railway (Backend)
