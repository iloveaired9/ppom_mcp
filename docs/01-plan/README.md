# Plan (계획) - Phase 1

## 목적
프로젝트의 기능 요구사항, 범위, 우선순위를 정의하는 단계입니다.

## 포함 내용
- ✅ 기능 목록
- ✅ 사용자 스토리
- ✅ 우선순위 (우선순위 높음/중간/낮음)
- ✅ 범위 (In-Scope / Out-of-Scope)
- ✅ 완료 기준

## 파일 구조
```
01-plan/
├── README.md (이 파일)
├── PLAN.md (실제 계획 문서)
└── feature-{name}.md (기능별 상세 계획)
```

## 다음 단계
Plan 완료 후 → Design 단계로 진행 (02-design/)

## 참고

- 각 단계는 독립적이지만 순차적으로 진행
